# 功能开关模块使用说明（feature_gate）

本文说明可选 **`feature_gate`** 模块（`CONFIG_FEATURE_GATE_MODULE`）：按命名槽位与 **license 字符串比对** 解锁高级能力（**非安全 DRM**）。事件 **72**。

**要求：** `CONFIG_EVENT_MAX_TYPES > 72`

**相关**：[47-远程运维模块使用说明.md](47-远程运维模块使用说明.md) · [43-配置方案对比指南.md](../40-应用开发/43-配置方案对比指南.md)

---

## 1. 启用

```bash
west build -b <board> . -- -DEXTRA_CONF_FILE=conf/features/feature_gate.conf
```

多 SKU 示例（网关型）：

```bash
west build -b nucleo_l4r5zi . -- \
  -DEXTRA_CONF_FILE="conf/features/scale.conf;conf/profiles/sku_gateway.conf"
```

---

## 2. 功能槽位

| 名称 | 说明 | 需 license |
|------|------|------------|
| `core` | 基础能力 | 否 |
| `cloud` | 云相关（`CONFIG_FEATURE_GATE_SLOT_CLOUD`） | 是 |
| `remote` | 远程运维（`CONFIG_FEATURE_GATE_SLOT_REMOTE`） | 是 |

Kconfig：

- `CONFIG_FEATURE_GATE_LICENSE` — 期望令牌
- `CONFIG_FEATURE_GATE_BOOT_LICENSE` — 启动时自动应用（量产可写入 OEM 令牌）

---

## 3. API

```c
int  feature_gate_apply_license(const char* token);
bool feature_gate_license_valid(void);
bool feature_gate_is_enabled(const char* feature_name);
```

`feature_gate_apply_license()` 返回 0 仅表示调用成功；token 是否有效请用 `feature_gate_license_valid()`。未 `init` 时返回 `APP_ERR_INIT`。

业务模块在启动远程能力前应检查：

```c
if (feature_gate_is_enabled(FEATURE_GATE_NAME_REMOTE)) {
    remote_ops_export_diag();
}
```

---

## 4. 测试

```bash
west build -b native_sim tests/ --build-dir build_tests_feature_gate -p always \
  -- -DCONF_FILE="prj.conf;prj_feature_gate.conf"
west build -t run --build-dir build_tests_feature_gate
```

---

## 5. 量产注意

- 字符串比对 **不能** 替代密码学授权；高安全场景请换 PSA / 签名校验后端。
- 各 SKU 通过 `conf/profiles/sku_*.conf` 覆盖 `FEATURE_GATE_LICENSE` 与 `PROVISIONING_DEVICE_ID`。
